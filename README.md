## Installation

*Pukunui* requires the libraries below:

- PortAudio

## Usage

Place *Pukunui* to the path of ASDF, and load it.

Then type `(pukunui:start)` start playing some noise. If you want to stop the noise, type `(pukunui:stop)`.
